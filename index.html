常见问题

Zero Forwarder 是如何实现带宽聚合的？ +
Zero Forwarder 利用多路径 TCP (MPTCP) 原理，在应用层实现了智能分包与重组。它可以同时利用多条物理链路或多个服务器节点的带宽，将单一连接的数据流拆分并行传输，从而实现带宽叠加效果。
部署 Zero Forwarder 需要什么样的硬件配置？ +
得益于 Rust 的高效内存管理，Zero Forwarder 对硬件要求极低。单核 CPU 和 512MB 内存即可轻松处理千兆级流量。对于万兆级应用，建议配置双核 CPU 和 4GB 以上内存。
数据传输是否安全？ +
非常安全。我们支持多种加密协议（如 TLS 1.3, Noise Protocol），并内置了防重放攻击和抗探测机制。您的数据在传输过程中全程加密，确保隐私不被窥探。
是否支持个人用户免费试用？ +
是的，我们提供功能受限的永久免费个人版，包含 5 个转发规则和基础带宽聚合功能。您可以随时下载并安装体验。
如果遇到问题如何获取支持？ +
您可以通过我们的官方文档中心查找解决方案，或者加入我们的 Discord 社区寻求帮助。付费用户还可以享受优先的邮件工单支持服务。
