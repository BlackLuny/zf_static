安装指南

Zero Forwarder 提供多种安装方式，请根据你的系统环境选择最适合的安装方法。

系统要求

  • 操作系统：Linux、macOS、Windows
  • 架构：x86_64、ARM64
  • 内存：最低 128MB，推荐 512MB+
  • 网络：需要互联网连接进行初始化

一键安装脚本

推荐使用一键安装脚本，适用于大多数 Linux 和 macOS 系统：

bash <(curl -sL https://get.zeroforwarder.com/install.sh)
提示：安装脚本会自动检测你的系统架构并下载对应版本。

手动安装

Linux

从 GitHub Releases 下载预编译的二进制文件：

# 下载最新版本
wget https://github.com/zeroforwarder/zeroforwarder/releases/latest/download/zeroforwarder-linux-amd64.tar.gz

# 解压
tar -xzf zeroforwarder-linux-amd64.tar.gz

# 移动到系统路径
sudo mv zeroforwarder /usr/local/bin/

# 验证安装
zeroforwarder --version

macOS

使用 Homebrew 安装（推荐）：

# 添加 tap
brew tap zeroforwarder/tap

# 安装
brew install zeroforwarder

# 验证安装
zeroforwarder --version

或手动下载安装：

# 下载 macOS 版本
wget https://github.com/zeroforwarder/zeroforwarder/releases/latest/download/zeroforwarder-darwin-amd64.tar.gz

# 解压并安装
tar -xzf zeroforwarder-darwin-amd64.tar.gz
sudo mv zeroforwarder /usr/local/bin/

Windows

从 GitHub Releases 页面下载 Windows 版本：

  1. 访问 Releases 页面
  2. 下载 zeroforwarder-windows-amd64.exe
  3. 将文件重命名为 zeroforwarder.exe
  4. 将文件移动到 PATH 环境变量包含的目录中

Docker 安装

使用 Docker 运行 Zero Forwarder：

# 拉取镜像
docker pull zeroforwarder/zeroforwarder:latest

# 运行容器
docker run -d --name zeroforwarder \
  -p 8080:8080 \
  -v /path/to/config:/etc/zeroforwarder \
  zeroforwarder/zeroforwarder:latest

从源码编译

如果你需要自定义编译或使用最新的开发版本：

注意：从源码编译需要 Rust 1.70+ 和相关构建工具。 
# 克隆仓库
git clone https://github.com/zeroforwarder/zeroforwarder.git
cd zeroforwarder

# 编译
cargo build --release

# 安装
cargo install --path .

验证安装

安装完成后，运行以下命令验证：

zeroforwarder --version

应该看到类似以下的输出：

Zero Forwarder v1.0.0
Built with Rust 1.70.0
